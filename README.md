# Real-Time Quick Commerce Analytics Dashboard (Blinkit-Inspired)

## Overview
This project features a Blinkit-inspired real-time quick commerce analytics dashboard built using Power BI. The dashboard analyzes sales performance, customer satisfaction, inventory distribution, and outlet efficiency to support data-driven business decision making.

## Business Objective
To conduct a comprehensive analysis of quick commerce sales data by tracking key performance indicators and visualizing trends across outlet types, locations, product categories, and customer ratings.

## Key KPIs
- Total Sales
- Average Sales
- Number of Items Sold
- Average Customer Rating

## Dashboard Features
- Interactive KPI cards for real-time performance monitoring
- Sales analysis by fat content, item type, outlet size, and outlet location
- Outlet establishment trend analysis using time-series visualization
- Dynamic filters for outlet location, size, and item type
- Drill-down insights for operational and strategic analysis

## Tools and Technologies
- Power BI
- Power Query
- DAX
- Excel

## Skills Demonstrated
Data Analysis, Data Cleaning, Data Modeling, DAX Calculations, KPI Reporting, Dashboard Development, Business Intelligence
